**My Notes**<br>
My Notes is a simple **CRUD** notes app developed in Kotlin. It allows users to create, edit, and delete notes. The app uses Room Database for efficient and reliable local storage of notes.<br>

**Features**<br>
**Create Notes**: Easily add new notes with a title and content.<br>
**Edit Notes**: Modify existing notes at any time.<br>
**Delete Notes**: Remove notes that are no longer needed.<br>
